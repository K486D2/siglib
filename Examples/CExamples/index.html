SigLib Example Programs

This directory contains the ANSI C source example programs for SigLib.

All of these examples can be compiled with the free Microsoft Visual Studio and the GCC (Cygwin, Linux or OS-X) C/C++ compilers using Gnuplot/C which is available from: https://www.numerix-dsp.com/files.

Application Folders

Individual Files

This folder includes shell scripts to assist with the recompilation of these examples under Linux and OSX:

gb.bat - Build an example using the Gnu compiler
gbr.bat - Build and run an example using the Gnu compiler
gbuildall.bat - Build all the examples using the Gnu compiler
grunall.bat - Execute all of the examples

This folder includes batch files to assist with the recompilation of these examples under DOS/Windows:

mb.bat - Build an example using the Microsoft compiler
mbr.bat - Build and run an example using the Microsoft compiler
mbuildall.bat - Build all the examples using the Microsoft compiler
mrunall.bat - Execute all of the examples
